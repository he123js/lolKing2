# 英雄联盟胜率排行榜

一个基于 Vue 3 的现代化英雄联盟胜率追踪工具，帮助你实时追踪游戏战绩，查看好友胜率排名。

## 🎮 功能特性

### 📊 核心功能
- **实时排名** - 按胜率自动排序，展示好友排名
- **数据管理** - 添加、编辑、删除游戏记录
- **智能搜索** - 快速搜索特定游戏ID
- **胜率可视化** - 彩色进度条展示胜率百分比
- **数据持久化** - 使用 localStorage 保存数据

### 🎯 界面特色
- **蓝白主题** - 英雄联盟风格的现代化界面
- **响应式设计** - 完美适配各种设备
- **动画效果** - 流畅的交互体验
- **直观操作** - 清晰的按钮和交互设计

## 🚀 技术栈

- **框架**: Vue 3 + Composition API
- **UI 组件**: Naive UI
- **构建工具**: Vite
- **状态管理**: Vue 响应式系统
- **数据存储**: localStorage

## 📦 快速开始

### 环境要求
- Node.js >= 18.3.0
- npm >= 7.0.0



### 构建生产版本

## 📱 使用说明

### 添加游戏记录
1. 点击"添加记录"按钮
2. 输入游戏ID、总场次和胜场数
3. 系统自动计算胜率并保存

### 编辑记录
1. 在记录列表中点击"编辑"按钮
2. 修改相关信息
3. 保存更新

### 搜索功能
- 在搜索框输入游戏ID关键词
- 系统自动筛选匹配的记录

## 🎨 界面预览

### 主界面
- 蓝白渐变背景设计
- 醒目的标题区域
- 搜索框和添加按钮
- 胜率排行榜表格

### 排行榜表格
- 排名序号（不同等级有不同颜色）
- 游戏ID和等级标签（大神/普通/新手）
- 胜场、总场次显示
- 胜率进度条可视化
- 操作按钮（编辑/删除）

### 弹窗功能
- 添加记录弹窗
- 编辑记录弹窗
- 表单验证和错误提示

## 🔧 数据格式

每个游戏记录包含以下字段：
```javascript
{
  id: "唯一标识",
  name: "游戏ID",
  wins: 胜场数,
  totalGames: 总场次,
  winRate: 胜率百分比
}
```

## 🌟 特色功能

### 智能验证
- 胜场数不能大于总场次
- 游戏ID不能重复
- 必填字段验证

### 用户体验
- 操作成功提示
- 错误信息友好显示
- 数据自动保存
- 响应式布局

## 📊 统计信息

系统会自动计算：
- 总记录数量
- 总游戏场次
- 平均胜率
